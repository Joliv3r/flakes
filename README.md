# Flakes

Here are some programs I sometimes want to run but requires dependecies I do not have or do not want to have system-wide.
The idea is that these flakes are standalone such that one only need to run `nix run github:joliv3r/flakes#<program>` to start the program.

The collection of programs will be very limited to obscure programs I sometimes want to use and are able to write a very simple flake for as my flake skills are quite limited.
